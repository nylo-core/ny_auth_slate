# Auth Slate
Generates a boilerplate for authentication for Nylo projects.

### Getting Started #
In your flutter project add the dependency:

``` dart 
dependencies:
  ...
  ny_auth_slate: ^0.0.1
```

## Requirements
* Dart >= 2.19.0
* Flutter >= 3.7.0

## Files generated:
- Pages
  - Login
  - Register
- Widgets
  - Buttons
  - Logo
  - Text Fields

## Usage

Run the below command in your project.

``` bash
flutter pub run lo_auth:main publish:all
```
